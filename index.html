Posted by Laura Jones on Monday, june 21st 2027

All modern websites and web applications are built using three fundamental technologies:HTML,CSS and JavaScript.There are the language of the web.
In this post, let's focus on HTML.We will learn what HTML is all about,and why you too should learn it.

What is HTML?

HTML stands for HyperText Markup Language. It's a markup language that developpers use to structure and describe the content of a webpage (not a programmming language).
HTML consists of elements that descibe differenttypes of content:paragraph, links, images, video, etc.Web browsers understand HTML and render HTML code as websites.
In HTML, each element is made up of 3 parts:

  • The opening tag
  • The closing tag
  • the actual element
